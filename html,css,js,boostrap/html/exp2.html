JavaScript Variables can be declared in 4 ways:Automatically
Using varUsing let,
Using const,
LET : let is a keyword.
let used as alternative of var keyword. by using let we can create local var or global var
LET :
used for creating a variable with fixed datatype.
Syn: LET varname=value;
ex: let a=10;
datatype of a is number, and we can't change datatype in rest of program.

Const
>const is a keyword.
>const kw used for creating a constant variables.means once we assign any value to variable, we can't change.
Syn: constvarname=value;
ex: const a=10;
we can't change value of a var.
Note: while using const keyword, don't use var& let keywords.
